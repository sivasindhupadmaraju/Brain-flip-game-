# 🧠 BrainFlip – Memory Card Game

**BrainFlip** is a premium, high-fidelity memory card game built with **React**, **Vite**, and **Vanilla CSS**. It features a robust, endless progression system across three difficulty tiers, each supporting hundreds of individual stages with historical stat tracking.

🚀 **[Play the Game Now!](https://brainflip-rose.vercel.app)**

## ✨ Features

- **🏆 Endless Progression**: Face hundreds of stages across Easy, Medium, and Hard tracks.
- **🗺️ Stage Selection Map**: Navigate through your unlocked stages on a beautiful, glass-morphic grid.
- **📊 Performance Tracking**: Hover over any cleared stage to view your past best score, moves, and time.
- **🌗 Dynamic Themes**: Toggle between deep space dark mode and clean light mode.
- **🎭 Playful Interaction**: Enjoy dynamic feedback texts for matches and misses, with Gen Z-inspired (and classic!) personality types.
- **📱 Responsive Design**: Fully optimized for mobile, tablet, and desktop play.

## 🛠️ Built With

- **Framework**: React 18
- **Bundler**: Vite
- **Styling**: Modern CSS (Variables, Glassmorphism, Animations)
- **State Management**: React Hooks + LocalStorage for persistent progress.

## 📦 Getting Started

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/sivasindhupadmaraju/Brain-flip-game-.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run locally**:
    ```bash
    npm run dev
    ```
